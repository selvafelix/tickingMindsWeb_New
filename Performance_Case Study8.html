Case Study: Performance Validation for Leading Global Hotel Wholesaler


Client Background

    Client is a leading global hotel wholesaler with core business focus on supplying hotel accommodation to the travel trade. Client collaborates with thousands of international hotels to offer an outstanding selection of global hotel accommodation.


Business Context

    Client had performance issues on their core travel portal which caters to their world wide customers. They wanted to improve the performance by upgrading the underlying hardware, but were not sure as to what extent they needed to invest on the infrastructure.


How did we help the client?

1.  Analyzed the application architecture, strategized performance testing for the portal, conducted performance test to measure the performance of the application.

2.  Identified that with the existing application design and capacity of the infrastructure application throughput was degrading.

3.  Nailed down performance issues at the database layer, made performance improvement strategies at the design level in the database layer level.


Benefits Delivered

1.  Avoided further spend on client’s end on infrastructure for pre-production and production instance.

2.  Application performance issues rectified by leveraging application recommendations above, which improved application performance significantly.

3.  Avoid end user dissatisfaction by averting performance issues.
