CASE STUDIES

Performance Testing

Case Study: Performance testing of a distribution management system for a leading ERP solutions provider


Client Background

 Client is leading solution development organisation in the ERP spar. Some of their products in their distribution management space have been implemented by the leading names in retail and telecom space. They were migrating their distribution management platform from a legacy to a web based three tier platform, in order to meet the increased demand.


Business Context

 Client had migrated their application and data from the legacy platform to a state of the art 3 tier platform. This application was intended for end clients from various geographies to mange their order books, manage inventory and generate invoices. Invoice generation was a major process in this ERP application suite as it affected various modules like inventory, client, sales man, sales routes etc. Their existing legacy application was failing them in this regard wherein an end user used to spend extended hours to generate invoices. The client wanted to see how the migrated application performed against the legacy application. While this was the expectation, the client had no access to the business data and hence the test data was expected to be generated in a synthetic fashion.


How did we help the client?

1.  Understood the ERP application suite, came up with critical business scenarios that were key for carrying out the performance testing.

2.  Understood the database structure and the relationships between the field in different tables.

3.  Developed excel macros to generate synthetic data to the tune of 125 GB of data, and pumped that data into SQL Server 2008 R2 DB.

4.  Used the open source solution JMeter to script the critical business scenarios Executed the performance test scenarios in a distributed fashion for more than 500 users generating more than 15000 invoices in an hour of load testing

5.  Detected various issues with the indexes, the way coding was done in certain areas which in turn was taking up huge application server time and DB server time.

6.  Helped the development team to identify bottlenecks and the areas where the optimisation needs to be carried out.


Benefits Delivered

1.  The invoice generation time was reduced from 10+ seconds to under 2 seconds, which was a considerable improvement while comparing the performance with the legacy system

2.  Carried out the end to end performance testing, ramping up to more than 1000 concurrent users all using open source solution, thereby saving the commercial tool license cost amounting to around $50K.

3.  Unearthing performance issues early in lifecycle, using a component and end to end test strategy.

4.  Synthetic test data was generated using excel macros, thereby saving on tool license cost for the test data management solutions, amounting to approximately $100K.
